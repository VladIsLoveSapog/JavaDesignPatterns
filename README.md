   Шаблоны проектирования программного обеспечения
</h1>

<p align="center">
   Обучающие материалы по паттернам GoF и основам промышленной разработки на Java для студентов 2-го курса
</p>

---

## О проекте

- Теоретические материалы и примеры кода по каждому паттерну GoF
- Основы Java, Spring (IoC, DI, AOP) и многопоточности
- Лабораторные работы с требованиями к сдаче

## Требования

- **JDK 17+**

## Рекомендации

Перед началом изучения курса рекомендуется:

- Ознакомиться с **объектно-ориентированным программированием** и знать его основные принципы
- Иметь представление о принципах **SOLID**
