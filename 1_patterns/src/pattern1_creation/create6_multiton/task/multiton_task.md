# Задание

В этом задании вам предстоит реализовать паттерн проектирования **"Мультитон" (Multiton)** для системы экспорта документов
в разные форматы: **PDF**, **CSV** и **XML**. Каждый экспортёр отвечает за преобразование содержимого документа в
соответствующий формат. Паттерн Мультитон гарантирует, что для каждого формата существует ровно один экземпляр экспортёра.

## Требования

1. Создать перечисление ```ExportFormat```: Содержит значения: ```PDF```, ```CSV```, ```XML```.
2. Создать интерфейс ```DocumentExporter```: Метод: ```String export(String content)```.
3. Создать классы ```PdfExporter```, ```CsvExporter``` и ```XmlExporter```, реализующие интерфейс ```DocumentExporter```:
    - Каждый класс должен иметь конструктор без параметров.
    - Метод ```export(String content)``` должен возвращать строку в формате: ```[FORMAT] content```, где ```FORMAT``` —
      имя соответствующего формата. Например, ```PdfExporter.export("hello")``` → ```"[PDF] hello"```.
4. Создать класс ```DocumentExporterMultiton```, реализующий паттерн Мультитон:
    - Использует ```EnumMap<ExportFormat, DocumentExporter>``` для хранения экземпляров экспортёров.
    - Предоставляет статический метод ```getExporter(ExportFormat format)```, возвращающий соответствующий экспортёр.
