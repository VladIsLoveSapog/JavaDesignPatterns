# Задание

Вы разрабатываете систему для представления меню ресторана. Меню состоит из категорий (например, «Супы», «Основные блюда»,
«Десерты»), каждая из которых может содержать как отдельные блюда, так и подкатегории (например, «Десерты» → «Горячие
десерты», «Мороженое»). Необходимо создать модель, которая позволит единообразно работать с любым элементом меню —
отображать его, вычислять суммарную калорийность и среднюю цену.

## Требования

1. **Определение интерфейса компонента:**
   - Создайте интерфейс ```MenuComponent```, который определяет общие методы для всех элементов меню:
     - ```String getName()``` — название элемента
     - ```void print(String indent)``` — вывод элемента с отступом
     - ```double getPrice()``` — цена (для блюда — своя, для категории — сумма всех блюд)
     - ```int getCalories()``` — калорийность (для блюда — своя, для категории — сумма всех блюд)
2. **Реализация листа (Leaf):**
   - Создайте класс ```MenuItem```, представляющий отдельное блюдо. Класс реализует ```MenuComponent``` и содержит: название, цену, калорийность.
3. **Реализация композита (Composite):**
   - Создайте класс ```MenuCategory```, представляющий категорию меню. Класс реализует ```MenuComponent``` и содержит список дочерних компонентов (```MenuItem``` и другие ```MenuCategory```).
4. **Дополнительные функции:**
   - Реализуйте методы ```add(MenuComponent)``` и ```remove(MenuComponent)``` в классе ```MenuCategory```.
   - Реализуйте метод для подсчёта средней цены блюда в категории (с учётом вложенных подкатегорий).

Для проверки используйте класс [RestaurantMain](RestaurantMain.java)
