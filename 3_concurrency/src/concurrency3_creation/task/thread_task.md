# Задание

Напишите Java-программу, которая создаёт и запускает несколько потоков с использованием интерфейса `Runnable`.

## Требования

1. Создайте класс `PrintTask`, реализующий `Runnable`, с полем `taskName`.
2. В методе `run()`: цикл от 1 до 5, между итерациями — случайная пауза 100–400 мс
   (`ThreadLocalRandom.current().nextLong(100, 400)`).
3. Создайте **один** экземпляр `PrintTask` и передайте его трём разным потокам `Thread`.
4. Запустите все три потока.

## Ожидаемое поведение

Порядок строк **непредсказуем** и меняется от запуска к запуску — это и есть признак
настоящего параллелизма. Не пытайтесь добиться фиксированного вывода.
